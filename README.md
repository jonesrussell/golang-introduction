# Go Fundamentals & Best Practices

This repository contains a record of the planning of this video tutorial series for https://www.youtube.com/@fullstackdev42

Also includes the code examples, assignments, cheat sheets, deployable encapsulated interactive tutorial with embedded videos, code playgrounds, etc.

## Structure

1. Go Basics: Variables, Types, and Control Flow for Beginners
2. Go Structs: Definition, Initialization, and Methods
3. Struct Embedding and Composition in Go
4. Pointers in Go: When to Use *Type vs Type
5. Understanding Go Interfaces: From Theory to Real-World Usage
6. Error Handling in Go: Patterns, Wrapping, and Custom Errors
7. Go Concurrency Explained: Goroutines and Channels
8. Go Slices and Maps: Internals and Best Practices
9. Dependency Injection in Go: Writing Testable, Maintainable Code
10. Avoiding Common Go Anti-Patterns (context.Value, global state, etc.)
11. Structured Logging with Zap: Production-Ready Logging Practices
12. Building CLI Tools in Go: Cobra and Flag Packages
13. Go Packages and Modules: Organization and Visibility
