# Instructor Notes: Zap Basics

## Teaching Techniques
- Show Zap usage
- Demonstrate structured logging
- Show log levels
- Emphasize: "Zap is fast and structured"

## Key Emphasis
- **Structured fields**: Key-value pairs
- **Log levels**: Debug, Info, Warn, Error
- **Fast**: Zero-allocation in many cases
- **Structured**: Easy to parse and search

## Common Questions
- "How do I use Zap?" - Create logger, use methods
- "What are the log levels?" - Debug, Info, Warn, Error
- "Is it fast?" - Yes, very fast

## Engagement
- "Notice how structured this is"
- "Easy to parse and search"
- "Zap is production-ready"

## Real-World Context
- Zap is widely used
- Fast and structured
- Production-ready
