# Instructor Notes: Go Modules

## Teaching Techniques
- Show module creation
- Demonstrate dependency management
- Show versioning
- Emphasize: "Modules manage dependencies"

## Key Emphasis
- **go.mod**: Module definition file
- **go.sum**: Checksums for dependencies
- **Versioning**: Semantic versioning
- **Dependencies**: `go get` to add, `go mod tidy` to clean

## Common Questions
- "What is a module?" - Collection of packages with version
- "How do I create one?" - `go mod init`
- "How do I add dependencies?" - `go get package`

## Engagement
- "Notice how modules manage dependencies"
- "Versioning is automatic"
- "This makes dependency management easy"

## Real-World Context
- Modules are standard now
- Replaced GOPATH
- Makes dependency management simple
