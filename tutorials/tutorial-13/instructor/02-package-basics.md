# Instructor Notes: Package Basics

## Teaching Techniques
- Show package declaration
- Demonstrate package structure
- Show imports
- Emphasize: "Packages organize code"

## Key Emphasis
- **Package declaration**: `package name`
- **Package name**: Should match directory (usually)
- **Imports**: Import packages you use
- **Exports**: Capitalized names are exported

## Common Questions
- "What is a package?" - Collection of related files
- "How do I name packages?" - Usually match directory name
- "What gets exported?" - Capitalized names

## Engagement
- "Notice how packages organize code"
- "Exports are simple: capitalize the name"
- "This is the foundation of Go organization"

## Real-World Context
- Packages are fundamental
- Used everywhere in Go
- Simple but powerful
