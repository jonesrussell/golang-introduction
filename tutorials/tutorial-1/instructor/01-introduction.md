# Instructor Notes: Introduction

## Screen Setup
- Code editor on left (80% screen)
- Terminal on right for output
- Use large font (18-20pt minimum)
- Dark theme for reduced eye strain

## Teaching Approach
- Welcome viewers warmly
- Set expectations for what they'll learn
- Show enthusiasm for Go's simplicity
- Briefly mention real-world Go usage (Docker, Kubernetes, etc.)

## Engagement Tips
- Ask: "Have you tried other languages before?"
- Mention that Go was designed to be simple to learn
- Promise hands-on coding throughout

## Key Points to Emphasize
- Go compiles fast
- Go produces single binaries (easy deployment)
- Strong typing catches bugs early
