# Introduction

**Duration:** 2-3 minutes

## Topics to cover:
- Welcome and what viewers will learn
- Why Go? Brief mention of [simplicity](https://go.dev/doc/faq#What_is_the_purpose_of_the_project), [performance](https://go.dev/doc/faq#Why_is_Go_fast), and [use cases](https://go.dev/solutions/)
- Show the final code we'll build (simple program that demonstrates all concepts)
- Setup check: [Go installed](https://go.dev/doc/install), editor ready ([VS Code](https://code.visualstudio.com/docs/languages/go)/Cursor recommended)

## Key teaching points:
- Go is [simple yet powerful](https://go.dev/doc/faq#principles)
- Great for [backend services](https://go.dev/solutions/webdev), [CLIs](https://go.dev/solutions/clis), and [cloud infrastructure](https://go.dev/solutions/cloud)
- [Fast compilation](https://go.dev/doc/faq#Why_does_Go_compile_so_fast) and execution
- [Strong standard library](https://pkg.go.dev/std)
