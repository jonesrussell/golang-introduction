# Instructor Notes: Cobra Introduction

## Teaching Techniques
- Show Cobra basics
- Demonstrate command structure
- Show subcommands
- Emphasize: "Cobra is powerful for complex tools"

## Key Emphasis
- **Commands**: Represent actions
- **Flags**: Command-specific options
- **Subcommands**: Nested commands
- **Powerful**: Handles complex CLIs

## Common Questions
- "When do I use Cobra?" - For complex tools with subcommands
- "How do I create commands?" - Use cobra.Command
- "What's the structure?" - Root command with subcommands

## Engagement
- "Notice how Cobra handles structure"
- "Subcommands make complex tools manageable"
- "Cobra is widely used"

## Real-World Context
- Cobra is standard for complex CLIs
- Used by kubectl, docker, etc.
- Powerful and flexible

- "Let's see subcommands in detail..."
