# Instructor Notes: Flag Package

## Teaching Techniques
- Show flag package usage
- Demonstrate different flag types
- Show parsing
- Emphasize: "Flag package is simple and powerful"

## Key Emphasis
- **Flag types**: String, int, bool, duration
- **Parsing**: flag.Parse() after defining flags
- **Usage**: flag.Usage() for help
- **Simple**: Good for simple tools

## Common Questions
- "How do I define flags?" - Use flag.String, flag.Int, etc.
- "When do I parse?" - After defining all flags, before using
- "How do I show help?" - flag.Usage() or -h/-help

## Engagement
- "Notice how simple this is"
- "Flag package is built-in"
- "Perfect for simple tools"

## Real-World Context
- Flag package is standard library
- Good for simple tools
- Cobra for complex tools
