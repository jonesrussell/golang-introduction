# Instructor Notes: Practical Example

## Teaching Techniques
- Build incrementally
- Show complete CLI tool
- Demonstrate all features
- Show proper structure

## Build Order
1. Set up Cobra structure
2. Create root command
3. Add subcommands
4. Add flags to commands
5. Integrate Viper for config
6. Add functionality
7. Show help and usage

## Live Commentary
- "Notice how we structure the commands"
- "Subcommands organize functionality"
- "Viper handles configuration"
- "This demonstrates all the patterns"

## Things to Emphasize
- Command structure
- Subcommands
- Flag handling
- Configuration management

## Engagement
- "What commands would you add?"
- "How would you organize this?"
- Challenge: "Add more subcommands"

## Variations to Mention
- Could add more commands
- Could add more configuration options
- Could add plugins
- Could add completion
